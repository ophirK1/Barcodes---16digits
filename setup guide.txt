1.להתקין מערכת רספברי חדשה.
2. להגדיר רשת למכשיר
3. להתקין את הספריה Barcodes במיקום /home/admin
4. פתיחת CMD
5. התקנת הספריות של הקוד
א. sudo apt-get update
ב. sudo apt-get install python3-pip
ג. sudo pip install pyusb --break-system-packages
ד. sudo apt install mpg321 -y
ה. sudo chmod +x /home/admin/Barcodes/main.py
6. התקנה של הפעלה באיתחול
	א. sudo crontab -e
		* 1 nano editor
	ב. לרשום הכי למטה בקובץ
		@reboot (sleep 10; /usr/bin/python3 /home/admin/Barcodes/main.py >> /home/admin/Barcodes/cron.log 2>&1) & 
		ctrl + O ואז Enter בשביל לשמור
		ctrl + X כדי לצאת

אם חלק ממערכת של יותר משער אחד יש להגדיר את כתובת האינטרנט של השרת ל
192.168.0.60\24
והשערים האחרים
192.168.0.XXX\24
7. sudo nano /etc/asound.conf
א. לרשום בקובץ
pcm.!default {
    type plug
    slave.pcm "hw:Headphones"
}

ctl.!default {
    type hw
    card "Headphones"
}

8.לוודא שwireless אינטרנט כבוי
9. להגביר סאונד עד הסוף
10. העברה למצב Console
	א. sudo raspi-config
	ב. System Options 1
	ג. S5 Boot
	ד. Console
10. אתחול
	sudo reboot
